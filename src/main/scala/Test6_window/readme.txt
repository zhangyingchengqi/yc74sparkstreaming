需求:   
	在实际生产中，与窗口相关的应用场景很常见，如电商每间隔10分钟统计某一商品前30分钟内累计销售总额，车站每隔一小时统计前3小时内的客流量等。 
	此类需求要用到streaming中的window来实现. 
	window应用涉及到批处理时间间隔，窗口时间间隔与滑动时间间隔。 
	
	监听网络中某节点上指定端口输出的数据流。 每10秒统计前30秒各单词累计出现的次数。 本地实现.
